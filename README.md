# 🚪 بازساز - Amiran Door Web App

کاتالوگ درب‌های ضد سرقت با قابلیت تلاش AR

## 🎯 ویژگی‌ها

- 📸 آپلود عکس درب خانه
- 🎨 پیش‌نمایش درب‌های مختلف
- 📱 رابط کاربری موبایل‌فرست
- 💾 ذخیره اطلاعات در Supabase

## 🛠️ نصب و اجرا

```bash
npm install
npm run dev
```

## 📦 تک‌نولوژی

- React 18
- Vite
- Tailwind CSS
- Supabase

## 📝 توضیحات

پروژه بر اساس معماری پاکیزه و بدون تعقیدات اضافی ساخته شده است.
